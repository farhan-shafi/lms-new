# LMS Documentation

Welcome to the Learning Management System (LMS) documentation. This repository contains comprehensive guides for both users and developers.

## Available Documentation

### [User Guide](./user_guide/README.md)

The User Guide provides detailed instructions for using the LMS system, including:

- Getting started with the system
- User role-specific guides (Admin, Instructor, Student)
- Course and lesson management
- Analytics and progress tracking features
- Troubleshooting common issues

This guide is intended for end-users of the LMS platform.

### [Developer Guide](./developer_guide/README.md)

The Developer Guide contains technical documentation for developers working on the LMS codebase, including:

- System architecture overview
- Technology stack details
- Directory structure
- Database schema
- Core components documentation
- Analytics & progress tracking implementation
- API documentation
- Guidelines for extending the system

This guide is intended for developers maintaining or extending the LMS platform.
